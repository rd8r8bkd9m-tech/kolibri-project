## 14. FREQUENTLY ASKED QUESTIONS

### Q: What does AGPL mean for my project?

**A:** If you use Kolibri (AGPL) in your project and deploy it on a server where users access it over a network (web, API, etc.), you must make your source code available.

### Q: Can I use AGPL code commercially?

**A:** Yes, but only if you also open-source your modifications under AGPL.

### Q: What if I don't want to open-source?

**A:** Use Commercial License ($10K+/year).

### Q: Can I modify AGPL code?

**A:** Yes, but modifications must also be AGPL.

### Q: What about linking AGPL library?

**A:** If you link against AGPL library, your program also becomes AGPL.

### Q: Can I charge for AGPL software?

**A:** Yes, but source must be available to recipients.

### Q: What's the difference between AGPL and GPL?

**A:** AGPL covers network use (SaaS loophole closed), GPL doesn't.
