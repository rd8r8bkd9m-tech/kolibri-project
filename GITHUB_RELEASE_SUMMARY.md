## ğŸ“ Technical Details

### Architecture Layers
1. **Core Engine (C/WASM)** - Decimal cognition logic
2. **Compression (JavaScript)** - DECIMAL10X algorithm
3. **Storage (Express.js)** - REST API
4. **UI (React/TypeScript)** - Web interface

### API Endpoints
```
POST   /api/auth/register         - User registration
POST   /api/auth/login            - User login
POST   /api/storage/upload        - File upload (auto-compress)
GET    /api/storage/list          - List files
GET    /api/storage/download/:id  - Download file (auto-decompress)
DELETE /api/storage/delete/:id    - Delete file
GET    /api/storage/info          - Storage statistics
POST   /api/health                - Health check
```
